# Quickstart


:::tip Note

Esta es una version inicial del flujo completo para poder forma a la documentacion

:::

### 1 Crear usuario

Primero que nada necesitamos crear un usuario. para esto lo crearemos con el endpoint [Create User](/api#/operations/PostV1User)


### 2 Enviar OTP

Generaremos una contraseña temporal para verificar al usuario con el enpoint [Create OTP](api#/operations/PostV1AuthOtp)

### 3 Verificar OTP

Verificaremos que el OTP sea correcto y obtendremos un token con el endpoint [Login](api#/operations/PostV1AuthSignIn) 

### 4 Actualizar contraseña 

Actualizaremos la contraseña con el sigueinte enpoint [change password](api#/operations/PutV1AuthChangePassword)

### 5 Crear tienda

crearemos la tienda para el usuario con el endpoint [Create Store](api#/operations/PostV1Store)

el objeto address y files los contenidos de deben ser url de archivos y para subir archivos tenemos el enpoint para subirlos [Subir archivos](api#/operations/PostV1FileUploadpublic)
y la url a guardar sera el **path** de la respuesta al guardar el archivo
l
### 6 Obtener Perfil

Obtendremos la informacion del usuario actual con el siguiente endpoint [Profile](api#/operations/GetV1AuthProfile)
